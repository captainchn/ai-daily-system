# 贡献指南

感谢你愿意为 AI Daily System 贡献力量！

## 如何贡献

### 报告问题

- 使用 [GitHub Issues](https://github.com/captainchn/ai-daily-system/issues) 报告 Bug 或功能需求
- 报告问题时，请尽量包含：
  - 清晰的标题和描述
  - 复现步骤
  - 环境信息（操作系统、Python 版本等）
  - 相关的日志或截图

### 提交代码

1. **Fork** 本仓库
2. 创建你的特性分支：`git checkout -b feature/your-feature-name`
3. 提交你的改动：`git commit -m 'Add some feature'`
4. 推送到分支：`git push origin feature/your-feature-name`
5. 创建 Pull Request

### 代码规范

- Python 代码遵循 PEP 8
- 提交信息使用清晰的中文描述
- 确保代码有适当的注释

## 项目结构
